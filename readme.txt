Apple-style mega menu for WordPress Block Themes. Fullwidth hover panels, smooth transitions, mobile accordion.

== Description ==

Sengeku Nav enhances the native WordPress Navigation block with Apple-style interactions. No replacement — just visual enhancement.

= Desktop =
* Fullwidth dropdown panels on hover
* Smooth fade transitions
* Focus-within for keyboard accessibility

= Mobile =
* Fullscreen blur overlay
* Tap-to-toggle accordion submenus
* Theme-colored with CSS custom properties

== Installation ==

1. Upload to `/wp-content/plugins/sengeku-nav/`
2. Activate
3. Your existing Navigation block gets enhanced automatically
